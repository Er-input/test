Требования к задаче:
1. Проверить поиск на сайте wikipedia.org. 
2. На странице "Просмотра кода" проверить наличие рабочей ссылки для возврата к "Заглавной странице".
3. Статья состоит из параграфов.
4. Любая статья должна иметь содержание.
5. Сменить язык на странице.

Requirements:
Для работы следует загрузить проект в среде разработки Intellij IDEA CE.
Установить Chrome Browser (в моем случае версия 114.0.5735.134).
Загрузить ChromeDriver (в моем случае версия 114.0.5735.90 ) версии как у установленного Chrome Browser.

Ход проведения тестирования:
1. Для проведения тестирования необходимо пройти по пути src/test/java в директории проекта.
2. Выбрать в среде разработки файл WikiTest.java.
3. Нажать на кнопку запуска возле метода Search.
4. Ожидать выполнения тестирования.

Структура проекта:
1. BaseTest.java - файл формирования основного класса тестирования для открытия и закрытия браузера.
2. MainPage.java - файл работы с заглавной страницей сайта wikipedia.org, в частности выполнение поиска статьи.
3. SubPage.java  - файл работы с страницей полученной после поиска статьи.
4. WikiTest.java - файл выполнения тестирования по заявленным требованиям.
